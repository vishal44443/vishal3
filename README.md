# Cozmoh — Template Repository

This is a static, single-page repository to quickly view and copy WordPress template links for client sharing.
It's designed for the Cozmoh internship  task.

## Structure
- index.html — main page
- css/style.css — styles
- js/app.js — data + small JS for copying links
- screenshots/ — 20 placeholder thumbnails (replace with real screenshots)
- README.md — this file

## How to use
1. Update `templateLink` values in `js/app.js` with actual WordPress template links.
2. Open `index.html` in a browser. No server required.
